PDFfit2 - real space structure refinement program.


PDFfit2 is normally installed as a part of DiffPy library.
To install or update the PDFfit2 package only:

    python setup.py install

PDFfit2 requires:
    python-devel        development files for Python
    gsl-devel           GNU Scientific Library

PDFfit2 installs the following executables:
    pdffit2             PDFfit2 launcher


For more information please visit the project web-page:

    http://www.diffpy.org
