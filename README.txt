PDFfit2 - real space structure refinement program.

PDFfit2 is normally installed as a part of DiffPy library.
To install or update the PDFfit2 package only:

    python setup.py install         # installs as diffpy.pdffit2

PDFFit2 requires development files for Python and GNU Scientific Library. 
These are available as Fedora packages python-devel and gsl-devel.

For more information please visit the project web-page:

    http://danse.us/trac/diffraction

or email Prof. Simon Billinge at billinge@pa.msu.edu
